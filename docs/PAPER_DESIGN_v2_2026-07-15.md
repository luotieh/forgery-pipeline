## 1. 问题与任务边界

- 局部扩散编辑视觉高度一致，传统 splicing 线索（边界、频率、相机指纹）被削弱。
- **设定为盲逆问题**：测试时 base 图不可观测（区别于 EditTrack 的有参考重编辑设定）；prompt 语义内容、seed 不在参数空间内（分别出界与边缘化）；CFG/steps 为 nuisance（主数据集抖动采样，PATCH 9.1）。
- **单算子假设**：编辑发生一次。multi-turn（DiffSeg30k 已在测）在本模型下无定义 → limitation 点名，可选 50 组二次编辑 probe。
- **不走多域融合路线**（不变）；跨生成器泛化理论留第二篇（不变）。
- **本篇任务**：一个框架内完成检测 + 像素级定位 + 算子归因——不止真假，还有"在哪、怎么编辑的、编辑到什么程度（序数意义）"。

## 2. 核心创新：score 残差场 + 算子参数逆估计（可辨识性刻画形态）

### 2.1 统一信号 — 多σ Tweedie 残差场 + 轨迹方向结构

冻结 **SD1.5**（v1 的 SD2 表述作废）latent 先验，K=5 个噪声级（单σ基线 t=50）：
- 幅值通道：`r_ε(t) = ε − ε̂_θ(z_t, t)`，`r_x(t) = z0 − ẑ0(t)`（Tweedie 一步估计）——**双通道分离，不坍缩**。
- **方向/相位通道（v2 新增，已确证 E3）**：相邻级 ε 残差方向余弦 `d_k = cos∠(r_ε(t_k), r_ε(t_{k+1}))` + 逐级 ε/x 比值——读取去噪**轨迹几何**而非幅值。
- 实现：17 维幅值统计 + 19 维方向/相位 = 36 维剖面；空间残差图保留给定位头。
- Tweedie 读法不变：残差是各尺度上 score 的估计，取证场有概率语义。
- 效率主张（C8）：少数 UNet 前向/图，对照反演式方法（DIRE ~15.9s、ZeroFake ~82s/图）。

### 2.2 三项输出（措辞按附录 A）

- **检测 / 定位**：不变；定位 = 恢复坐标 M。
- **算子归因（核心）**：类型识别 = 在**纤维化**参数空间 (t0, c, M) 中定位编辑——c 决定坐标存在性（t0 仅在强度可变纤维上有定义，指令式 t0≡T；全图算子 M 退化）。估计器条件式：先判纤维、再读类条件坐标。
- **t0 坐标的确证形态（E4/E5）**："强序数恢复 + 粗桶强度分级 + 分辨率刻画"，**不称点估计/逆估计**；框架级"算子参数的逆估计"为许可命名（描述映射方向与输出空间性质）。
- **区别于 softmax 的两个可证伪预言**：已见强度间插值（Phase E 组合泛化）+ 混淆几何随参数距离（E13）。

### 2.3 机制（非定理）——现在一条已闭合、一条已预注册

- **(i) SDEdit 真实性–忠实性权衡 → 强度签名**：**已闭合**——单调签名（ρ=0.700）与其自预言的失效模式（饱和，AUC 0.798→0.578、双条件 s\*）同被证实。写法定调："理论命中了自己的边界"，饱和是发现不是失效。
- **(ii) RePaint 逐步投影 → 机制/空间签名**：边界不连续 + 内部在流形上；**互补覆盖**——(ii) 恰在 (i) 的重生成汇合区幸存，幸存判别量是空间性的。方向性预言（空间头 > 全局幅值 probe 0.48）已写入 prereg v3，gate2 裁决。
- "扩散难分"解释：(t0, c) 区域重叠 → 签名重叠；混淆–参数距离图（E13）为其直接检验。

## 3. 方法设计（与 method 稿 §3 同步）

- 冻结打分先验：**单 SD1.5**，VAE fp32 / UNet fp16；scorer–generator 解耦由 Test-B 压测。
- 四头一场：H1 检测、H2 定位（latent→pixel 对齐集中解决）、H3 条件式 g_ψ（强度支输出**校准单调分数**，只以序数/粗桶读出；ordinal 损失，ℓ2 入消融）、H4 可靠性（可选）。
- grounding 一致性 = 同一坐标 M 被 H2/H3 各读一遍的自洽，非附加正则。
- **可辨识性感知读出（v2 新增小节，= method §3.5）**：等可分性分桶（校准分位数，绝不均匀切 s 轴）、分辨率剖面 AUC(k,k+1)、s\*、对照常数预测器的校准 MAE。
- 待决架构项：E12 归因后决定是否"按头冻结特征"（t0 优化的方向特征不默认全头共享）。

## 4. 子创新（v2 重排；每条带理论钩子与证据位）

1. **多σ Tweedie 残差场**（幅值主干）。理论：Tweedie + score 匹配。证据：E1 + E2（nuisance 鲁棒为新增卖点）。差异化：对 Snap-Back（2025.11，多强度像素级感知曲线、仅检测）——本文是 latent score 场 + 方向结构 + 定位/归因 + 机制解释；"多尺度曲线"一词可让，"score 轨迹几何"不让；Snap-Back 式 LPIPS 曲线特征列为消融对照。
2. **编辑算子参数逆估计框架 + 可辨识性刻画**（全场最原创，形态更新）。理论：SDEdit 权衡（已闭合）+ RePaint 投影（已预注册）。产出：纤维化参数空间、条件式估计、证据梯子、双条件饱和刻画。
3. **方向/相位特征**（v2 升格具名）。理论钩子：去噪轨迹几何；CFG/steps 扰动轨迹几何 → 增量部分收缩仍显著，与机制自洽（E3）。
4. **可辨识性刻画协议**（v2 新增，方法贡献）：预注册证据梯子（序数/粗分级/信息等效/点估计）+ group-fold/cluster-CI 协议 + 机械 verdict。卖点：读出粒度由测得可辨识性决定；MAE 门未过即不用其词——审稿最难攻击的一节。
5. **跨生成器轻量轴**：gate3 PASS（掉点 0.03–0.04）+ 先验集成缓解；结论入册以 vae_rt FPR 复核为前提（E11）；理论留第二篇（不变）。
- **出局**：v1 子创新 3（流形/雅可比定位）——无实验、无排期、算力不允许；降为 future work 一段，multiLID 差异化随之移入该段。

## 5. 与现有工作的关系（v2 骨架）

- **生成参数反演谱系（新设小节，本文的家谱）**：model parsing（Asnani，TPAMI'23，从图像反推架构/损失）→ 推理超参可检测性（Xu et al. ECCV'24：步数 10 类 26–57%、seed ~99%；编辑面积–归因掉点曲线为 9.2b 分层依据）→ 编辑归因（EditTrack：**有参考**、模型 ID、重编辑策略 / DiffSeg30k：盲、逐像素、编辑器 ID）→ **本文：盲、单图、参数空间粒度（机制 + 序数强度）**。递进天然成立；"attribution of AI editing" 词已被占，引言措辞避让。
- **多 timestep / 反演检测线**：Snap-Back、ESIDE（多 timestep 集成）、ANL（单步噪声估计特征 ≈ r_ε 通道既有版本）、Temporal Score Analysis（相邻 timestep score 动态定位伪影——离方向特征最近，必引）、FakeInversion / ZeroFake / DIRE（成本与跨范式失效数字支撑 C8 与 LaMa 边界实验）。
- **定位与数据集**：DiffSeg30k 列为**外部评测集**（Phase D 增轴：在其上跑定位 + 编辑器/算子识别，外部效度）；GRE、EditScout、REALEDIT（真人编辑域，Test-D 候选）。
- **强度信号旁证**：水印脆弱性文献的"信号衰减 vs 编辑强度"单调曲线 = t0 签名的主动侧证据，且独立解释高端饱和。
- 保留：DIRE/AEROBLADE/FIRE/DRCT 差异化、MiraGe/CRDA/Community Forensics 归第二篇。

## 6. 诚实的风险与边界（v2 登记）

- t0 措辞禁令（E5）与"固定 CFG/steps"脚注（E7；0.092 对 0.10，边距如实给出；(7.5,30) 分解待清，越线则主动升 limitation）。
- E12：gate0 det 掉点未归因——入册前必须解释；可能引出按头特征冻结。
- E11：gate3/4 好数字在 PATCH 7 的 vae_rt 复核前不写成结论（压缩链混淆本身生成器不变，会伪装成"跨生成器稳"）。
- c 轴（E8）：全文档措辞最保守的坐标——至今无独立变化源，IP2P 入数据前不做任何 c 轴强主张。
- 单算子假设 / multi-turn：limitation 点名（对照 DiffSeg30k）。
- 竞争窗口：Snap-Back 与 DiffSeg30k 均为 2025 Q4——多σ检测与编辑器识别两个邻位已有人落子，**执行速度即护城河**；gate2 之后的 Phase B–D 不再有可拖延的决策点。
- 保留红线：无 provable；"操作 vs 模型指纹"由 gate2 cross-model 解耦；score 推理成本如实报告。
